Employee Portal.

It is MEVN application that is implemented using Vue cli, Vue Router and single file components.
The app is responsive from mobile to desktop. Also includes Vue transitions. REST Api has been created.

Technologies Used:
VueJS, NodeJS, CSS3

Instructions: (Docker required)

1. Go to project folder using the change directory (cd) command in the terminal.
2. Run the command:
	docker-compose run
3. To view the front-end, type: localhost:8002 in the url.
4. To view the back-end routes, type: localhost:3002 in the url
5. For mongoDB, type: localhost:8003 in URL.
